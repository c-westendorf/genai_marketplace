## Using Skills

Skills are workflows with steps, checkpoints, and decision points. Use them when:
- Tackling a new workflow for the first time
- You need a structured process to follow
- You're onboarding new team members

**Reference in conversation:**
```

@eda-workflow.md
```

Or copy the skill file to your project and follow step-by-step.
